OTPublisherKitDelegate Protocol Reference

Conforms to NSObject
Declared in OTPublisherKit.h

Overview

Used for sending messages for an OTPublisher instance. The OTPublisher class includes a delegate property. When you send the [OTPublisherKit initWithDelegate:] message or the [OTPublisherKit initWithDelegate:name:] message, you specify an OTSessionDelegate object.
